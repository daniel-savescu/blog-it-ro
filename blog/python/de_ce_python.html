De ce Python?

Python este un limbaj bun cu scop general, la nivel înalt. Designul său îl face foarte lizibil, ceea ce este mai important decât pare. Fiecare program de calculator este scris numai o dată, dar citit și revizuit de multe ori, adesea de mulți oameni. Fiind lizibil, de asemenea, este mai ușor de învățat și de reținut, prin urmare mai ușor de scris. În comparație cu alte limbi populare, Python are o curbă de învățare blândă care te face să fii productiv mai devreme, dar are adâncimi pe care le poți explora pe măsură ce câștigi experiență. Concisitatea relativă a lui Python vă face posibil să scrieți un program care este mult mai mic decât echivalentul său într-un limbaj static. Studiile au arătat că programatorii tind să producă aproximativ același număr de linii de cod pe zi – indiferent de limbaj – așa că, scrierea jumătate din liniile de cod vă dublează productivitatea, exact așa. Python este arma nu atât de secretă a multor companii care cred că acest lucru este important.

PyPython este cel mai popular limbaj pentru cursurile introductive de informatică la cele mai bune colegii americane. Este, de asemenea, cel mai popular limbaj pentru evaluarea pro- abilitate de gramatică de peste două mii de angajatori. Și, desigur, este gratuit, ca în bere și vorbire. Scrie orice vrei cu Python, și folosește-l oriunde, în mod liber. Nimeni nu vă poate citi programul Python și spune: „Este un mic program frumos pe care il ai acolo. Ar fi păcat dacă i s-ar întâmpla ceva.”

Python rulează aproape peste tot și are „baterii incluse” – o mulțime metrică de software util în biblioteca sa standard. Dar, poate cel mai bun motiv pentru a folosi Python este unul neașteptat: oamenilor le place în general. De fapt, le place să programeze cu el, mai degrabă decât să-l trateze doar ca pe un alt instrument pentru a face lucrurile. Adesea, ei vor spune că le lipsește o funcție a Python atunci când trebuie să lucreze cu un alt limbaj. Și asta este ceea ce separă Python de majoritatea a semenilor săi.
