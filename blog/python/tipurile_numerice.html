Tipuri de date numerice în Python

Python are două tipuri de date principale pentru stocarea numerelor:

  1. Numere întregi (int):

    • Reprezintă numere întregi, pozitive, negative sau zero.

    • Nu au zecimale.

    • Se scriu fără ghilimele, de exemplu: 5, -100, 2048.

    • Pot fi foarte mari, limitate doar de memoria calculatorului.

    • Operații posibile: +, -, *, /, // (împărțire cu rest), %, ** (exponentiere).

    • Funcții utile: int(), pow().

    Exemplu:

    var_int = 123
    print(var_int) #Output : 123

    rezultat = 10 + 5 * 2
    print(rezultat) #Output : 20

  2. Numere reale (float):

    • Reprezintă numere cu zecimale.

    • Se scriu cu sau fără ghilimele, de exemplu: 3.14, -9.25, 10.5e2 (10.5 la puterea 2).

    • Pot fi foarte mari sau foarte mici, dar pot avea o precizie limitată.

    • Operații posibile: +, -, *, /, // (împărțire cu rest), %, **.

    • Funcții utile: float(), round().

    Exemplu:

    var_float = 15.67
    print(var_float) #Output : 15.67

    suma = 3.14 + 4.2
    produs = 5 * 2.5
    print(suma, produs) #Output : 7.34 12.5

Observații:

  • Python detectează automat tipul de date numeric la atribuire.

  • Se poate converti un număr dintr-un tip în altul folosind funcțiile specifice (int(), float()).

  • Este important să alegeți tipul de date adecvat pentru operațiile dorite. De exemplu, dacă aveți nevoie de o precizie mare, folosiți float.

  • Există și un al treilea tip de date numeric, mai puțin utilizat: complex. Acesta stochează numere complexe formate din partea reală și partea imaginară.
