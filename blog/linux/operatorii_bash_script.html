Ce este Bash Scripting?

Bash scripturile folosesc o varietate de operatori pentru a efectua calcule, compara valori, controla fluxul și manipula lanțuri de caractere. Iată o descriere detaliată a diferitelor tipuri de operatori:

  1. Operatori aritmetici

    • Efectuați calcule matematice simple pe numere întregi.

    • Exemple: +, -, *, /, % (modulo)

    • Exemplu: var=$((x + y))

  2. Operatori relaționali

    • Comparați valorile pentru a determina adevărul sau falsul unei condiții.

    • Exemple: == (egal), != (diferit), <, >, <=, >=

    • Exemplu: if [ $x -gt $y ]; then echo "X este mai mare decât Y"; fi

  3. Operatori logici

    • Combinați condiții pentru a crea expresii logice mai complexe.

    • Exemple: && (AND), || (OR), ! (NOT)

    • Exemplu: if [[ $file -f && $file -r ]]; then echo "Fișierul există și este lizibil"; fi

  4. Operatori pentru şiruri

    • Manipulați lanțuri de caractere.

    • Exemple: + (concatenare), == (egalitate), != (diferit)

    • Exemplu: nume_complet="$prenume $nume"

  5. Operatori de testare a fișierelor

    • Verificați proprietățile și starea fișierelor și directoarelor.

    • Exemple: -f (fișier regular), -d (director), -r (lizibil), -w (scriibil), -x (executabil)

    • Exemplu: if [[ -f $fisier ]]; then echo "Fișierul există"; fi

  6. Operatori bitwise

    • Efectuați operații pe biți individuali (utilizate mai rar în scripturi simple).

    • Exemple: & (AND bitwise), | (OR bitwise), ^ (XOR bitwise)

Pentru informații mai detaliate despre fiecare operator, inclusiv utilizarea și sintaxa specifică, puteți consulta documentația Bash sau căutați online [man bash].
