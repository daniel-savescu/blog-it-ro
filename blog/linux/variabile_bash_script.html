Ce sunt varibilele din Bash Script?

Variabilele din bash script sunt containere care stochează valori temporar în timpul rulării scriptului. Ele permit scriptului să manipuleze date și să păstreze informații între diferite părți ale codului.

Iată o descriere detaliată a variabilelor bash:

Tipuri de variabile

  • Variabile de mediu: Aceste variabile sunt definite la nivel de sistem și sunt disponibile pentru toate procesele. Ele stochează informații globale despre sistem, cum ar fi calea de acces curentă ($PATH), utilizatorul curent ($USER) și ID-ul de proces ($PPID).

  • Variabile locale: Aceste variabile sunt definite în cadrul unui script sau funcție și sunt disponibile doar în scopul respectiv. Ele nu afectează variabilele cu același nume definite în alte părți ale scriptului sau în mediul global.

Definirii de variabile

Pentru a defini o variabilă, utilizați sintaxa următoare:

name = value

  • name este numele variabilei, care trebuie să înceapă cu o literă sau o subliniere (_). Nu poate conține spații sau caractere speciale (cu excepția câtorva cazuri speciale).

  • value

    este valoarea pe care doriți să o stocați în variabilă. Poate fi un șir de text, un număr, o altă variabilă sau rezultatul unei comenzi.

Exemplu:

username="Ion_Cuza"

current_date=$(date +%d-%m-%Y)

echo "Salut, $username! Astazi este $current_date."

În acest exemplu:

  • Se definește o variabilă username cu valoarea "Ion_Cuza".

  • Se definește o variabilă current_date folosind comanda date pentru a obține data curentă formatată (%d-%m-%Y).

  • Se utilizează variabilele username și current_date în interiorul comenzii echo pentru a afișa un mesaj personalizat.

Accesarea valorilor variabilelor

Pentru a accesa valoarea unei variabile, utilizați prefixul $ urmat de numele variabilei. De exemplu:

message="Bash scripting este distractiv!"

echo $message

Eliminarea variabilelor

Pentru a elimina o variabilă și a elibera memoria pe care o folosește, utilizați comanda unset:

unset username

Rețineți

  • Bash nu necesită declararea tipului de date al variabilelor. Variabilele pot stoca diferite tipuri de date în același script.

  • Puteți efectua operații aritmetice pe variabile care conțin valori numerice.

  • Bash acceptă operatori de șir pentru manipularea șirurilor de text stocate în variabile.
