Top comenzi esențiale Linux

Cunoașterea câtorva comenzi esențiale poate transforma modul în care interacționezi cu Linux. Iată o descriere detaliată a celor mai importante comenzi:

  1. ls: Listarea fișierelor

    • Funcționalitate: Afișează lista fișierelor și subdirectoarelor din directorul curent.

    • Opțiuni utile:

      • -a: Afișează toate fișierele, inclusiv cele ascunse (încep cu ".").

      • -l: Afișează detalii suplimentare despre fișiere, cum ar fi permisiunile, data și ora modificării, proprietarul și dimensiunea.

      • -h: Afișează dimensiunea fișierelor în format uman-citit (ex: KB, MB, GB).

    • Exemplu: ls -lh afișează detalii complete despre fișierele din directorul curent, cu dimensiuni în format uman-citit.

  2. cd: Schimbarea directorului

    • Funcționalitate: Permite navigarea prin sistemul de fișiere prin schimbarea directorului curent.

    • Utilizare:

      • cd nume_director: Se mută la directorul specificat.

      • cd ..: Se mută cu un nivel în sus în ierarhia directoarelor.

      • cd ~: Se mută la directorul home al utilizatorului actual.

    • Exemplu: cd Documente/Proiect navighează la directorul "Proiect" din folderul "Documente".

  3. mkdir: Crearea de directoare

    • Funcționalitate: Permite crearea de directoare noi în sistemul de fișiere.

    • Utilizare: mkdir nume_director creează un director cu numele specificat.

    • Opțiuni utile:

      • -p: Permite crearea de directoare imbricate simultan (ex: mkdir -p /home/user/Proiect/Subfolder).

      • -m: Permite setarea modului (permisiunilor) pentru noul director.

    • Exemplu: mkdir -p Proiect/Subfolder creează directorul "Proiect" și subdirectorul "Subfolder" în interiorul lui.

  4. cp: Copierea fișierelor

    • Funcționalitate: Permite copierea fișierelor din locație în locație.

    • Utilizare: cp fișier_sursă fișier_destinație copiază fișierul "fișier_sursă" în locația specificată de "fișier_destinație".

    • Opțiuni utile:

      • -r: Permite copierea recursivă a directoarelor, inclusiv a subdirectoarelor și fișierelor din interior.

      • -v: Afișează informații detaliate despre procesul de copiere.

    • Exemplu: cp fisier.txt /home/user/Documente copiază fișierul "fisier.txt" în folderul "Documente" al utilizatorului.

  5. mv: Mutarea fișierelor

    • Funcționalitate: Permite mutarea fișierelor din locație în locație, schimbând locația originală.

    • Utilizare: mv fișier_sursă locație_destinație mută fișierul "fișier_sursă" în locația specificată de "locație_destinație".

    • Opțiuni utile:

      • -r: Permite mutarea recursivă a directoarelor.

      • -f: Forțează suprascrierea fișierelor existente cu același nume.

    • Exemplu: mv fisier.txt /home/user/Desktop mută fișierul "fisier.txt" pe desktopul utilizatorului.

  6. rm: Ștergerea fișierelor

    • Funcționalitate: Permite ștergerea permanentă a datelor. Folosiți cu atenție!

    • Opțuni importante:

      • -f: Forțează ștergerea (nu este recomandat).

      • -i: Solicită confirmare (folosiți întotdeauna!).

      • -v: Afișează informații detaliate.

      • -R: Șterge recursiv directoarele (periculos!).

      • -n: Simulează ștergerea (utilă pentru testare).

    • Exemple:

      • rm -i fisier.txt: Șterge "fisier.txt" cu confirmare.

      • rm -v *.txt: Șterge toate fișierele ".txt" cu detalii.

      • rm -R /home/user/Proiect: Șterge recursiv "Proiect" (extrem de periculos!).

  7. cat: Vizualizarea conținutului fișierelor

    • Funcționalitate: Permite afișarea conținutului fișierelor text pe ecran.

    • Utilizare: cat fișier afișează conținutul fișierului specificat.

    • Opțiuni utile:

      • -n: Afișează numerele de linie pentru fiecare linie din fișier.

      • -s: Ignoră caracterele de spațiu alb la sfârșitul fiecărei linii.

    • Exemplu: cat fisier.txt | grep "cuvant" afișează doar liniile din fișierul "fisier.txt" care conțin cuvântul "cuvant".

  8. grep: Căutarea de text în fișiere

    • Funcționalitate: Permite căutarea de text specific în interiorul fișierelor.

    • Utilizare: grep cuvant fișier caută cuvântul "cuvant" în fișierul specificat.

    • Opțiuni utile:

      • -i: Ignoră diferența dintre majuscule și minuscule.

      • -c: Afișează numărul de apariții ale cuvântului în fișier.

      • -n: Afișează numerele de linie unde apare cuvântul.

    • Exemplu: grep -c "eroare" log.txt numără de câte ori apare cuvântul "eroare" în fișierul "log.txt".

  9. sort: Sortarea liniilor unui fișier

    • Funcționalitate: Permite sortarea liniilor unui fișier text în ordine specifică.

    • Utilizare: sort fișier sortează liniile fișierului specificat în ordine alfabetică ascendentă.

    • Opțiuni utile:

      • -r: Inversează ordinea sortării (descendentă).

      • -n: Sortează numeric, pe baza valorilor numerice din linii.

      • -k: Specifică coloana pe baza căreia se face sortarea.

    • Exemplu: sort -k3 -n date.txt sortează fișierul "date.txt" numeric după a treia coloană.

  10. man: Vizualizarea manualelor de utilizare

    • Funcționalitate: Permite accesarea manualelor de utilizare pentru diverse comenzi și programe.

    • Utilizare: man command afișează manualul de utilizare pentru comanda specificată.

    • Exemplu: man ls afișează manualul de utilizare pentru comanda ls.

  11. history: Vizualizarea istoricului comenzilor

    • Funcționalitate: Permite vizualizarea și reexecutarea comenzilor introduse anterior.

    • Utilizare: history afișează lista comenzilor introduse anterior.

    • Opțiuni utile:

      • -c: Șterge istoricul comenzilor.

      • !!: Reexecută ultima comandă introdusă.

      • !n: Reexecută comanda cu numărul n din istoric.

  12. apt: Gestionarea pachetelor software (Debian/Ubuntu)

    • Funcționalitate: Permite instalarea, actualizarea și eliminarea pachetelor software din distribuțiile Debian/Ubuntu.

    • Utilizare:

      • sudo apt install nume_pachet: instalează pachetul software specificat.

      • sudo apt update: actualizează lista de pachete disponibile.

      • sudo apt upgrade: actualizează pachetele instalate la cea mai recentă versiune.

      • sudo apt remove nume_pachet: elimină pachetul software specificat.
