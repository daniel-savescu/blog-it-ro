Controlul fluxului execuției in Bash Script

Flow controlul într-un script Bash se referă la utilizarea diferitelor structuri de control pentru a gestiona direcția execuției codului în funcție de condiții specifice sau repetarea anumitor secvențe de cod. Cele mai comune structuri de control de flux în Bash sunt if, case, for, while și until. Să le examinăm în detaliu pe fiecare dintre acestea:

  1. Structura 'if'

    2. Structura 'if' este folosită pentru a executa o secvență de comenzi doar dacă o anumită condiție este îndeplinită.

    Sintaxa de bază:

    if [ condiție ]; then
    comenzi
fi

    Exemplu:

    #!/bin/bash

num=10

if [ $num -gt 5 ]; then
    echo "Numărul este mai mare decât 5"
fi

    if-else și if-elif-else:

    if [ condiție ]; then
    comenzi
else
    alte_comenzi
fi

    if [ condiție1 ]; then
    comenzi1
elif [ condiție2 ]; then
    comenzi2
else
    alte_comenzi
fi

    Exemplu if-elif-else:

    #!/bin/bash

num=10

if [ $num -gt 10 ]; then
    echo "Numărul este mai mare decât 10"
elif [ $num -eq 10 ]; then
    echo "Numărul este exact 10"
else
    echo "Numărul este mai mic decât 10"
fi
  2. Structura 'case'

    3. Structura 'case' este utilizată pentru a compara o variabilă împotriva mai multor modele și a executa diferite comenzi în funcție de rezultatul comparației.

    Sintaxa de bază:

    case variabilă in
    model1)
        comenzi1
        ;;
    model2)
        comenzi2
        ;;
    *)
        comenzi_default
        ;;
esac

    Exemplu:

    #!/bin/bash

fruit="apple"

case $fruit in
    "apple")
        echo "Este un măr"
        ;;
    "banana")
        echo "Este o banană"
        ;;
    *)
        echo "Fruct necunoscut"
        ;;
esac
  3. Structura 'for'

    4. Structura 'for' este utilizată pentru a itera printr-o listă de elemente sau un interval de numere.

    Sintaxa de bază:

    for variabilă in listă; do
    comenzi
done

    Exemplu:

    #!/bin/bash

for num in 1 2 3 4 5; do
    echo "Numărul este $num"
done

    Iterare prin fișiere:

    for file in /path/to/directory/*; do
    echo "Fișierul este $file"
done
  4. Structura 'while'

    5. Structura 'while' este utilizată pentru a executa comenzi în mod repetat cât timp o condiție este adevărată.

    Sintaxa de bază:

    while [ condiție ]; do
    comenzi
done

    Exemplu:

    #!/bin/bash

count=1

while [ $count -le 5 ]; do
    echo "Numărul este $count"
    count=$((count + 1))
done
  5. Structura 'until'

    6. Structura 'until' este similară cu while, dar execută comenzi până când condiția devine adevărată (adică cât timp condiția este falsă).

    Sintaxa de bază:

    until [ condiție ]; do
    comenzi
done

    Exemplu:

     #!/bin/bash

count=1

until [ $count -gt 5 ]; do
    echo "Numărul este $count"
    count=$((count + 1))
done

Concluzie

Utilizarea corectă a acestor structuri de control permite crearea de scripturi Bash eficiente și flexibile. Structurile 'if', 'case', 'for', 'while', și 'until' sunt esențiale pentru controlul fluxului execuției și permit luarea de decizii și repetarea de operații în funcție de condiții specifice.
