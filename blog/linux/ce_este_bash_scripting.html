Ce este Bash Scripting?

Un script Bash este un fișier text simplu care conține o serie de comenzi pe care le executa Bash, un interpret de linie de comandă popular pe Linux și macOS. Aceste comenzi pot fi orice ați introduce manual în terminal, cum ar fi ls pentru listarea fișierelor, cp pentru copierea fișierelor sau mv pentru mutarea fișierelor.

De ce să folosim bash scripting?

Există multe beneficii ale utilizării bash scripting-ului:

  • Automatizarea sarcinilor repetitive: Dacă efectuați frecvent aceleași comenzi la terminal, le puteți pune într-un script și rula scriptul o singură dată. Acest lucru vă economisește timp și reduce erorile.

  • Efectuarea sarcinilor complexe: Scripturile Bash pot efectua sarcini mai complexe decât este posibil cu o singură comandă. Puteți folosi logica condițională (de exemplu, if-else) și bucle (de exemplu, for, while) pentru a controla fluxul scriptului.

  • Crearea de programe personalizate: Scripturile Bash pot fi folosite pentru a crea programe simple personalizate care îndeplinesc nevoile specifice.

Cum să scrieți un script Bash

Pentru a scrie un script Bash, urmați acești pași:

  • Creați un fișier text: Utilizați un editor de text pentru a crea un fișier text nou.

  • Adăugați shebang-ul: Prima linie a scriptului trebuie să fie o shebang. Acesta specifică interpretul care trebuie utilizat pentru a rula scriptul. Pentru Bash, aceasta este de obicei #!/bin/bash.

  • Adăugați comenzile: Adăugați comenzile pe care doriți să le execute scriptul, câte una pe linie.

  • Salvați fișierul: Salvați fișierul cu o extensie .sh (de exemplu, myscript.sh).

  • Faceți scriptul executabil: Utilizați comanda chmod +x pentru a face scriptul executabil.

Exemplu de script Bash

Iată un exemplu simplu de script Bash care listează toate fișierele din directorul curent:

#!/bin/bash

# Listați toate fișierele din directorul curent
ls -l

# Afișați un mesaj
echo "S-au listat toate fișierele din directorul curent."

Pentru a rula acest script, salvați-l ca list_files.sh și apoi executați următoarea comandă în terminal:

./list_files.sh

Acest script va lista toate fișierele din directorul curent, urmat de mesajul "S-au listat toate fișierele din directorul curent."
